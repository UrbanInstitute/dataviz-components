# resizeObserver action

An action you can use to easily to check when a DOM element's dimensions change using the Resize Observer API. Sometimes is more efficient and less bug prone than Svelte's built-in `bind:clientWidth` approach.

## Usage

```js
<script>
  import { resizeObserver } from '@urbaninstitute/dataviz-components/actions';

  let width = 0;
</script>

<div use:resizeObserver="{(el) => (width = el.clientWidth)}">
  My width is: {width}
</div>
```
