## Usage

```js
import { AxisY } from "@urbaninstitute/dataviz-components";
```

## Want to update axis style straight from the Layercake repo?

Use this css instead:

```js
<style>
    .tick, .axis-label {
      font-size: 12px;
      font-family: var(--font-family-sans-serif);
    }

    .tick line {
      stroke: #dedddd;
    }
    .tick:first-of-type line {
      stroke: #000000;
    }
</style>
```
