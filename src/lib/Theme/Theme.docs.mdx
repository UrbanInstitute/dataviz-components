# Theme

A theme wrapper that provides shared styles for children components. These styles are primarily made up of a series of CSS variables that represent Urban's design system. They also includes some default typographic styling applied to html headings. Components in this library make use of these variables and styles and expect them to be defined, and won't display correctly outside of a parent `<Theme />` component.

All of the `Theme` styles are scoped and won't be applied to content outside of the `<Theme />` itself.

In order to display correctly, these components will also need access to Urban's fonts, which can be loaded with a [font component](/docs/theming-fonts--docs)

You get everything you need out of the box by importing and using these components like so:

```html
<script>
  import { Theme, FontsUrban, Button } from "@urbaninstitute/dataviz-components";
</script>
<FontsUrban />
<Theme>
  <Button />
</Theme>
````

## Stylesheet import

If you need access to the `<Theme>` component CSS as a global stylesheet, this library exports a copy that can be imported like this:

```javascript
import "@urbaninstitute/dataviz-components/style/theme";
```
